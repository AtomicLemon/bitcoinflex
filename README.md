Bitcoin Flex Core integration/staging repository
=====================================


Bitcoin Flex is the new Proof of Stake (POS) currency that teatures:

- Fast transactions.
- Low transaction fees
- Masternodes

More information at [bitcoinflex.org](http://www.bitcoinflex.org)
